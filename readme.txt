== Description ==

**Zenith Slider** is Responsive Highlights Content Slider that is easy to use and implement into any post, or page, by copy-pasting the generated shortcode.

View [Demo](http://themeflection.com/plugins/wordpress/zenith/index.php) 

[Plugin Home](http://themeflection.com/plug/responsive-wordpress-slider/)

For support visit my [support forum](http://themeflection.com/forums/forum/plugins/wordpress/)

Main Features:

*   Responsive
*   Easy to use
*   Fully Customizable
*   Modern Look
*   Simple integration via copy-paste Shortcode



== Installation ==

Zenith Slider can be installed in two different ways:

*  Via WordpPress uploader
*  or via FTP Client

1- Steps to follow to install plugin via WordPress uploader

*   go to WordPress admin panel and click on "plugins" from the dashboard menu.
*   Then from Plugins -> Add New -> Upload -> Choose file -> and then choose zenith-slider.zip and press Open-> Install Now -> Activate Plugin

2- Steps to follow to install plugin via FTP

*   Unzip the file
*   Upload unziped tf_numbers folder to your WordPress plugins directory (/wp-content/plugins/)
*   Activate from WordPress menu Plugins section
